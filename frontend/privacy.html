Privacy Policy

Your data privacy is important to me. Here's how I handle your information when you use PR Code Reviewer.

Overview

PR Code Reviewer is a Chrome extension designed to enhance your code reviews using AI-powered insights. This Privacy Policy explains how I handle your data and ensure your privacy when you use the extension.

Data Collection

I do not collect or store any personal data. All operations are performed locally in your browser, and any data processed for code analysis is sent directly to OpenAI's API using the API key you provide.

Use of OpenAI API

PR Code Reviewer relies on OpenAI APIs to analyze the content of your GitHub pull requests. When you use the extension:

  • Your code is transmitted to OpenAI's servers for processing and analysis.
  • Your OpenAI API key is stored locally in your browser and is not shared with me or anyone else.

Your Responsibility

As the user, you are responsible for ensuring the security and appropriate use of your OpenAI API key. Please review OpenAI’s Privacy Policy to understand how they handle your data.

Security

I take your security seriously:

  • Your API key and extension settings are stored securely in your browser's local storage.
  • None of your data is transmitted to my servers or shared with any third-party service other than OpenAI.

Changes to This Policy

I may update this Privacy Policy from time to time. If I make significant changes, I will provide a prominent update on the extension’s landing page. Please check this page periodically for updates.

Contact Me

If you have any questions, concerns, or suggestions about this Privacy Policy, feel free to reach out to me at pr-code-review@azzy.cloud.
