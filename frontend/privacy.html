Privacy Policy

Your privacy matters. Here's my simple, no-nonsense privacy policy.

Overview

PR Code Reviewer is an open-source Chrome extension that helps you review GitHub pull requests using AI. The extension runs entirely in your browser - there are no backend servers involved.

Data Handling

Here's what you need to know about how your data is handled:

  • No tracking, no analytics, no cookies - the extension doesn't collect any data
  • Your API keys and settings are stored only in your browser's local storage
  • When you review a PR, the selected code is sent only to your chosen AI provider's servers for analysis
  • Review results are stored temporarily in your browser's local storage and can be cleared anytime

Website Analytics

This landing page for PR Code Reviewer uses Umami, a privacy-focused, cookie-free analytics tool, to understand general website traffic trends. This tracking is completely anonymous and does not collect personal data, store cookies, or track users across websites.

The extension itself does not use any analytics or tracking tools.

API Integration

The extension uses various AI providers' APIs for code analysis. When you use the extension:

  • Your selected code is sent to your chosen AI provider for review using your provided API key
  • The extension supports multiple providers (OpenAI, DeepSeek, etc.) - please review their respective privacy policies:
    • OpenAI Privacy Policy
    • DeepSeek Privacy Policy

Important: Any code you select for review will be sent to your chosen AI provider's servers for analysis. If you have sensitive files or proprietary code, please ensure you have appropriate permissions before using the extension.

Open Source

This extension is open source - you can review the code, contribute, or report issues on GitHub. This transparency ensures you can verify exactly how your data is handled.

Questions?

If you have any questions about this privacy policy or the extension, feel free to:

  • Email me at pr-code-review@azzy.cloud
  • Open an issue on the GitHub repository
