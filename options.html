PR Code Reviewer Options

Overview

PR Code Reviewer is a Chrome extension for efficient GitHub pull request reviews. Powered by GPT-4, it analyzes code changes and suggests improvements or flags issues.

The extension formats new, modified, or removed code into prompts for GPT-4 to deliver precise feedback, speeding up the review process.

How It Works

The extension reviews code in the Files section of a GitHub PR, identifying additions, deletions, and edits, which it sends to GPT-4. Feedback is based only on PR changes, maintaining repository confidentiality.

For sensitive files, avoid using the extension to ensure privacy.

Disclaimer

Using PR Code Reviewer means code snippets from PRs are sent to OpenAI for analysis per their Privacy Policy. Ensure appropriate permissions before using it on sensitive code.
