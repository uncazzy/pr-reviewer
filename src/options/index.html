PR Options

Overview

PR Code Reviewer is a Chrome extension for efficient GitHub pull request reviews. Powered by advanced AI models, it provides detailed, contextual feedback on your code changes.

The extension formats new, modified, or removed code into prompts for AI analysis to deliver precise feedback, focusing on best practices, potential issues, and suggestions for improvement.

How It Works

When you click the extension icon on a GitHub PR page, it automatically identifies code additions, deletions, and edits, which it sends to the selected AI model. Feedback is based only on PR changes, maintaining repository privacy.

For sensitive files, avoid using the extension to ensure privacy.

Privacy & Security

Using PR Code Reviewer means code snippets from PRs are sent to your chosen AI provider for analysis per their respective privacy policies. Your API keys and settings are stored locally in your browser.
